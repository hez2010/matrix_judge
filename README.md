# RayJudge

A WIP new, unified and cross-platform Judge Platform for Online Judge.

## Targets

- cross-platform high-performance program executer
- possibility of custom judgers
- detailed specification of features
- 100% coverage
- GDPR-compliant
- easy to deploy
